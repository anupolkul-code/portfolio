# Anupol Kullasut – Portfolio

A personal portfolio website showcasing my projects, technical skills, and background as a Data Science & Software Innovation student at Ubon Ratchathani University.

## About

This is a static single-page portfolio built with plain HTML, CSS, and JavaScript — no frameworks required. It highlights my experience in full-stack web development, mobile development, and AI integration.

## Featured Projects

- **CUTCAMP** – A full-stack barbershop booking web application built with Python/Django, PostgreSQL, and Gemini API for AI-powered hairstyle recommendations.
- **CookConnect** – A recipe-sharing platform with role-based access control, full CRUD, and an admin analytics dashboard.
- **Plane of Bullet** – A 2D arcade game built with Python and Pygame using OOP principles.

## Tech Stack

- **Languages:** Python, Dart, JavaScript, Java, HTML, CSS
- **Frameworks:** Django, FastAPI, Flutter, Tailwind CSS, Bootstrap
- **Databases:** PostgreSQL, MySQL, Firebase
- **Tools:** Docker, Git, GitHub, Figma, Postman, n8n

## Structure

```
portfolio/
├── index.html        # Main HTML file
├── style.css         # Stylesheet
├── script.js         # JavaScript interactions
└── *.png / *.JPEG    # Project screenshot assets
```

## Running Locally

Open `index.html` directly in a browser, or serve it with a local HTTP server:

```bash
python -m http.server 3000
```
